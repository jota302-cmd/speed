# SpeedTest Lite ⚡

**SpeedTest Lite** (Speed) es un medidor de banda ancha ultra-ligero que mide tu velocidad de conexión sin scripts invasivos ni recolección masiva de datos.

## 🚀 Características

- **Rendimiento Puro**: Mide latencia, velocidad de bajada y subida al instante.
- **Sin Dependencias Pesadas**: Evita el bloatware presente en probadores comerciales.
- **Privacy-First**: No compartimos tu IP, ubicación ni resultados con terceros.
- **Responsive**: Funciona de forma fluida y elegante tanto en mobile como en desktop.

## 🛠️ Especificaciones Técnicas

- **Stack**: HTML5, Tailwind CSS, Vanilla JavaScript.
- **Despliegue**: Subdominio Vercel (`speed.optimicrotools.com`).
